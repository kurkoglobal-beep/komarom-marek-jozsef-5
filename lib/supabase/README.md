# Supabase konfiguráció

Ez a mappa kizárólag a későbbi Supabase-integráció helye.

Jelenleg:

- nincs Supabase kliens;
- nincs hálózati kapcsolat;
- nincs autentikáció;
- nincs adatbázis-művelet;
- nincs aktív projektazonosító.

A `config.ts` csak a két előkészített környezeti változó nevét központosítja.
